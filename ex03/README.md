# 作业3：使用 millis() 控制 SOS 信号

## 实验目标
使用 `millis()` 函数实现非阻塞方式控制 LED 产生 SOS 摩尔斯电码闪烁信息。

## SOS 信号规则
- **S（短闪）**：200ms 亮 + 200ms 灭，重复 3 次
- **O（长闪）**：600ms 亮 + 200ms 灭，重复 3 次
- **S（短闪）**：200ms 亮 + 200ms 灭，重复 3 次
- 播放完毕后停顿 2000ms 再循环

## 技术要点
- 使用时序表数组存储闪烁模式
- `millis()` 实现非阻塞状态机
- 串口输出当前播放状态

## 运行效果
LED 按照 SOS 信号规律闪烁：三短、三长、三短，循环播放。
